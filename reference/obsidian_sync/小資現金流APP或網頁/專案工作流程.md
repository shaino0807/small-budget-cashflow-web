## 踩坑筆記

- Firebase MCP 預設跳過。
- GitHub CLI token 目前失效，無法建立 repo 或 push。
- TWSE e添富頁面可查主檔、近期配息與資料日期；成分股權重、NAV 與折溢價需分投信官網處理，不能用第三方資料硬補。
- TWSE `STOCK_DAY` 可提供月內價格折線；目前已寫入 0056、00878、006208 共 21 筆。
- TWSE OpenAPI 可補上市股票主檔與每日行情；目前股票主檔 1090 檔，若要支援上櫃股票需再接 TPEx。
- 本機 Node 沒有 Playwright；已改用 Chrome headless 產出截圖與 runtime 檢查。Chrome headless 在此環境需要 `--disable-software-rasterizer --disable-gpu-compositing --in-process-gpu --no-sandbox` 才穩定。
- 元大/國泰官方頁多為 SPA 或動態 API 載入；直接 HTTP 回應不一定包含 NAV/持股表格，parser 必須把 loaded、partial、no_visible_rows、no_visible_nav 分清楚。
- 靜態部署不能讓瀏覽器直接寫回 `etf-database.json`；公開上線若要自動更新，需使用後端、serverless、GitHub Action 或排程產生資料快照。
- 前端穿透重疊度曾在初始化時太早讀取 `etfDatabase`，造成 TDZ runtime exception；已調整初始化順序並用 Chrome runtime 檢查確認 console error 0 / exception 0。
